Salesforce DevTools (v3.10.0)

リリースノート (2023/4/10)

  1. 新機能：Lightning上でも利用できる項目一括編集、コピー、削除機能をリリースしました。🎉

                今までClassic上しか利用できないカスタム項目一括編集機能をLightning上利用できるようになりました。
            
    さらに項目コピー時、項目レベルセキュリティ権限含めコピーされますので、項目権限追加設定が不要となります！
    
            オブジェクト跨ぎの項目コピー機能も従来通りオブジェクト指定するだけで簡単にコピーできてしまえます！

  2. SOQLクエリのエラー結果がより詳細に確認できます。

    SOQLクエリ実行エラー発生した場合、より詳細なエラー情報を表示できるようになりました。

  3. 項目説明をオブジェクト項目定義書に追加しました。

    多数の開発者から要望を受けた、項目説明と項目履歴をオブジェクト項目定義書に出力ができるようなりました。
    
            関連Issueはこちら : #56
            #71

  4. 他のSandboxに接続してしまう問題が対応しました。

    複数Sandboxで同時開くときに、他のSandboxに接続してしまう問題を対応しました、さらに現在利用しているユーザID情報が簡単に確認及び再認証ができます。
    
            関連Issueはこちら : #70
